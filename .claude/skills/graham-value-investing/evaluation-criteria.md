# Graham Value Investing - 评分标准详解

## Graham评分系统 (0-40分基础分)

### 内在价值折扣 (20分)

评估股票价格相对于内在价值的折扣程度。

| 安全边际范围 | 得分 | 说明 |
|-------------|------|------|
| ≥ 50% | 20分 | 极度低估，强烈买入机会 |
| ≥ 40% | 16分 | 显著低估，优秀买入机会 |
| ≥ 30% | 12分 | 合理低估，符合Graham标准 |
| ≥ 20% | 8分 | 轻度低估，可考虑买入 |
| ≥ 10% | 4分 | 小幅折扣，观望为主 |
| < 10% | 0分 | 折扣不足，不建议买入 |

**计算方法**：
```rust
let margin_of_safety = (intrinsic_value - current_price) / intrinsic_value;

let score = match margin_of_safety {
    m if m >= 0.50 => 20,
    m if m >= 0.40 => 16,
    m if m >= 0.30 => 12,
    m if m >= 0.20 => 8,
    m if m >= 0.10 => 4,
    _ => 0,
};
```

### 盈利质量 (10分)

评估EPS的稳定性和增长性。

| EPS增长率 | 得分 | 说明 |
|----------|------|------|
| > 10% (稳定) | 10分 | 优秀增长，盈利质量高 |
| 5-10% | 8分 | 稳健增长，盈利质量良好 |
| 0-5% | 6分 | 增长缓慢，但保持稳定 |
| -5% 至 0% | 3分 | 盈利波动，质量一般 |
| < -5% | 0分 | 盈利下降，质量差 |

**稳定增长定义**：
- 过去5年每年都正增长
- 标准差 < 平均值的30%
- 无极端负增长年份

**计算方法**：
```rust
// 计算历史增长率
let eps_growth = calculate_cagr(eps_history);

// 计算稳定性
let stability = calculate_stability(eps_history);

// 综合评分
let score = if stability && eps_growth > 0.10 {
    10
} else if eps_growth > 0.05 {
    8
} else if eps_growth > 0.0 {
    6
} else if eps_growth > -0.05 {
    3
} else {
    0
};
```

### 财务健康 (10分)

评估公司的财务稳健程度。

| 负债率 | 流动比率 | 得分 | 说明 |
|--------|----------|------|------|
| < 30% | > 2.0 | 10分 | 财务非常健康 |
| < 50% | > 1.5 | 7分 | 财务健康 |
| < 70% | > 1.0 | 4分 | 财务一般 |
| ≥ 70% | < 1.0 | 0分 | 财务风险高 |

**关键指标**：

1. **负债率 (Debt Ratio)**
   ```
   负债率 = 总负债 / 总资产
   ```
   - < 30%: 优秀
   - 30-50%: 良好
   - 50-70%: 一般
   - > 70%: 风险高

2. **流动比率 (Current Ratio)**
   ```
   流动比率 = 流动资产 / 流动负债
   ```
   - > 2.0: 优秀
   - 1.5-2.0: 良好
   - 1.0-1.5: 一般
   - < 1.0: 风险高

3. **利息保障倍数 (Interest Coverage)**
   ```
   利息保障倍数 = EBIT / 利息支出
   ```
   - > 5: 优秀
   - 3-5: 良好
   - 2-3: 一般
   - < 2: 风险高

## Buffett质量加分 (最高+7分)

### ROIC加分 (最高+2分)

| ROIC范围 | 加分 | 说明 |
|---------|------|------|
| > 20% | +2分 | 卓越资本回报 |
| 15-20% | +2分 | 优秀资本回报 |
| 10-15% | +1分 | 良好资本回报 |
| < 10% | +0分 | 资本回报一般 |

### ROE加分 (最高+2分)

| ROE范围 | 加分 | 说明 |
|---------|------|------|
| > 20% | +2分 | 卓越股东回报 |
| 15-20% | +2分 | 优秀股东回报 |
| 12-15% | +1分 | 良好股东回报 |
| < 12% | +0分 | 股东回报一般 |

### 护城河加分 (最高+3分)

| 护城河评分 | 加分 | 说明 |
|-----------|------|------|
| 3/3 | +3分 | 宽护城河，强竞争优势 |
| 2/3 | +2分 | 中等护城河 |
| 1/3 | +1分 | 窄护城河 |
| 0/3 | +0分 | 无明显护城河 |

**护城河评分标准**：

1. **品牌优势** (1分)
   - 知名品牌，消费者认知度高
   - 具有定价能力
   - 高客户忠诚度
   - 品牌难以复制

2. **成本优势** (1分)
   - 规模经济显著
   - 垂直整合
   - 专有技术或专利
   - 低成本供应商地位

3. **网络效应** (1分)
   - 用户增长带来价值提升
   - 平台效应明显
   - 高转换成本
   - 数据积累优势

## 总分计算

```rust
// Graham基础分
let graham_score = discount_score + quality_score + health_score; // 0-40分

// Buffett加分
let buffett_bonus = roic_bonus + roe_bonus + moat_bonus; // 0-7分

// 总分
let total_score = graham_score + buffett_bonus; // 0-47分
```

## 评分解释

| 总分范围 | 评级 | 投资建议 |
|---------|------|----------|
| 40-47 | AAA | 卓越价值投资机会，强烈买入 |
| 35-39 | AA | 优秀价值投资机会，买入 |
| 30-34 | A | 良好价值投资机会，买入 |
| 25-29 | BBB | 合理价值投资机会，考虑买入 |
| 20-24 | BB | 一般价值投资机会，观望 |
| 15-19 | B | 较弱价值投资机会，谨慎 |
| < 15 | CCC | 差价值投资机会，避免 |

## 评分使用建议

1. **综合考量**：不要仅依赖总分，要关注各维度得分
2. **趋势分析**：关注得分的变化趋势
3. **行业对比**：与同行业公司对比得分
4. **历史对比**：与公司历史得分对比
5. **风险调整**：根据风险偏好调整投资决策
